# 🚀 WORK SCHEDULER

Take control of your day with a fast, simple, and stylish daily planner.
**Work Scheduler** helps you organize your work hours, keep track of priorities, and stay productive without the clutter.

## 🌐 Live Demo
[Launch Work Scheduler](https://maxkarltun.github.io/WORK-SCHEDULER/)

## ✨ Features
- 🗓️ Displays the current day at the top of the planner
- ⏰ Hour-by-hour time blocks for standard business hours
- 🎨 Color-coded blocks to quickly show **past**, **present**, and **future**
- 📝 Add and edit tasks/events in each time block
- 💾 One-click save with `localStorage` persistence
- 🔄 Saved events remain after page refresh

## 👤 User Story
> **AS AN** employee with a busy schedule  
> **I WANT** to add important events to a daily planner  
> **SO THAT** I can manage my time effectively

## ✅ Acceptance Criteria
- Current day is displayed when the planner opens
- Time blocks for business hours are shown
- Each time block is color-coded by time status
- Events can be entered in each time block
- Clicking save stores event text in local storage
- Saved events persist after refreshing the page
